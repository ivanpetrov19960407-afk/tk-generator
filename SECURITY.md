# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 2.x | ✅ |
| 1.x | ⚠️ best effort |
| <1.0 | ❌ |

## Reporting a Vulnerability

Если вы нашли уязвимость, пожалуйста:

1. Не публикуйте детали в публичных issue.
2. Откройте приватный security advisory в GitHub (если доступно) или создайте issue с пометкой `security` без деталей эксплуатации.
3. Приложите минимальные шаги воспроизведения, затронутые версии и ожидаемый impact.

Мы стараемся подтвердить получение отчёта в течение 72 часов и дать план исправления после воспроизведения.
