# DXF import (v1)

Поддерживается только 2D DXF и извлечение базовых параметров детали.

## Поддерживаемые сущности
- Геометрия: `LWPOLYLINE`, `LINE`, `POLYLINE`.
- Размеры: `DIMENSION` (если есть числовое значение).
- Подсказки: `TEXT` / `MTEXT`, имена слоёв.

## Определение length/width
1. Для каждой геометрической сущности строится bbox.
2. В первой версии выбирается самый крупный bbox (по площади).
3. `length = max(dx, dy)`, `width = min(dx, dy)`.
4. Если есть минимум 2 валидных `DIMENSION`, они приоритетнее bbox.

При конфликте между `DIMENSION` и bbox в `meta.warnings` пишется предупреждение.

## Определение thickness (приоритет)
1. `options.thickness`.
2. CLI `--thickness`.
3. Имя файла (`/(\d+)\s*mm/i` и близкие варианты).
4. `TEXT` / `MTEXT`.
5. Имена слоёв (`T=30`, `T30`, `THK=30`, `30mm`).

Если найдено несколько значений, выбирается источник с более высоким приоритетом и пишется warning.

## Единицы
- Читается `$INSUNITS` из `HEADER`.
- `1` (inches) конвертируется в мм (`25.4`).
- `4` (millimeters) без конвертации.
- Если не задано/нераспознано — принимается мм и добавляется warning.

## Ограничения первой версии
- Только 2D DXF.
- Многодетальные файлы обрабатываются ограниченно: выбирается один основной контур.
- Толщина определяется только эвристиками и конвенциями в тексте/слоях/имени файла.
