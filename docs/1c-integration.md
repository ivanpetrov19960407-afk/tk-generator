# Интеграция с 1С: экспорт калькуляций

В `tk-generator` добавлен экспорт калькуляций в формат, удобный для импорта в 1С.

## CLI-флаги

- `--export-1c` — формирует XML-файл в формате «упрощённый CommerceML 2»:
  - путь: `output/1c_export_<YYYY-MM-DD>.xml`.
- `--export-1c-csv` — формирует CSV для упрощённого импорта:
  - путь: `output/1c_export_<YYYY-MM-DD>.csv`.

Оба флага можно использовать одновременно.

## Какие данные включаются

Экспорт строится на основе расчёта себестоимости (`calculateTotalCost`) и содержит:

1. **Номенклатура**
   - наименование;
   - артикул (используется `tk_number`);
   - единица измерения (`control_unit`, при отсутствии — `шт`).

2. **Калькуляция**
   - материалы;
   - ФОТ;
   - накладные;
   - прибыль;
   - итого (расчётная цена реализации).

3. **Спецификация**
   - операции;
   - нормы труда (`НормаТруда`);
   - нормы машинного времени (`НормаМаш`).

## Примеры

```bash
# XML экспорт для 1С
node src/index.js --input examples/batch_input.json --output output --export-1c

# CSV экспорт для упрощённого импорта
node src/index.js --input examples/batch_input.json --output output --export-1c-csv

# Сразу оба формата
node src/index.js --input examples/batch_input.json --output output --export-1c --export-1c-csv
```

## Примечания

- Для экспорта в 1С автоматически выполняется расчёт стоимости (даже без `--export-cost`).
- XML-файл имеет структуру, совместимую с обменом через XML-парсеры/обработки 1С и легко мапится в объекты `Номенклатура + Калькуляция + Спецификация`.
